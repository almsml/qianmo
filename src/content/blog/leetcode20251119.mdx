## Problem Analysis：
题目要我们找到origial的值,每次找到就要将origial的值*2,如果我们暴力对origial进行暴力遍历，也可以解出来，但是时间复杂度是O(n²)，我们可以转换思路，求数组中original的2^k的值，把它存到一个变量x里，最后我们只需要对变量进行倒数遍历，遇到第一个0就代表数组里没有那个值了。
## Code:
```astro
class Solution {
public:
    int findFinalValue(vector<int>& nums, int original) {
        for(int i=0;i<nums.size();i++){
            for(int j=0;j<nums.size();j++){
            if(nums[j]==original){
                original*=2;
                continue;
                }
            }
        }
        return original;
    }
};
```
